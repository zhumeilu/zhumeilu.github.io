关闭tomcat报错Cannot allocate memory

Sep 26, 2018 | tomcat

报错如下:

主要是系统内存不够,无法分配内存了

解决方案:
修改./bin/catalina.sh中的内存分配
将最小内存修改到系统剩余内存以内

JAVA_OPTS=" -server -Xms256M -Xmx2048M
