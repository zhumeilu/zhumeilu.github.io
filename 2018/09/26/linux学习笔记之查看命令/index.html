linux学习笔记之查看命令

Sep 26, 2018 | linux

cat命令

用于查看纯文本文件（内容较少的）
常用于较小的文件

more命令

more命令用于查看纯文本文件（内容较多的）

head命令

查看纯文本文档的前N行

less命令

查看纯文本文件（内容较多的）
常用
可以实时查询文件中关键字
向下搜索: /+关键字
向上搜索: ?+关键字
退出:q
下一页:page up 或者回车
上一页:page down
下一行:空格

tail命令

tailf动态显示文件内容,用于查看日志
tail -f 同上
tail -f -n 100 显示100行

grep命令

用于搜索日志内容
示例:grep -A10 -B10 --color '' xxx.log
    关键字向上保留10行,向下保留10行内容
-a 读取二进制文件
-i 忽略大小写
-p 递归查询该文件夹内的所有文件

wc命令

统计指定文本的行数、字数、字节数
-l 显示行数
-w 单词数
-c 字节数
